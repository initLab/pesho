# Pesho PCB

### Hardware description:

 * Raspberry Pi
 * 3 optoizolated relays
 * 3 optoizolated inputs
 * pesho's pcb

### Design

The design is done using latest version of KiCAD from the repository.
All modules, that weren't available are in pesho.pretty folder.
